# dotfiles

My prefs and settings.

## Dependencies

- [rcm]


## Installation

Clone the repository:

```sh
$ git clone git@github.com:andrewgarner/dotfiles.git ~/.dotfiles
```

Update and install dotfiles managed by [rcm]:

```sh
$ rcup
```
