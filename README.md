# dotfiles

My prefs and settings.


## Dependencies

- [GNU Stow](https://www.gnu.org/software/stow/)


## Installation

```
git clone --recurse-submodules https://github.com/andrewgarner/dotfiles.git ~/dotfiles
cd ~/dotfiles
./install.sh
```
